# Handoff: F1 赛道年代记（F1 知识 App 原型）

## Overview
一个 web-based、移动端优先的 F1 赛车知识 App。核心概念：**主屏是一条从 1950 年发车、蜿蜒到 2025 年的赛道**，76 个赛季全部落在路上（每年一个弯/节点）；年代是正文主线，名车/技术/车手作为「博物馆」知识库入口，可从任意条目定位回时间轴。灵感来自书籍《Formula 1 Technology》：第一章的历届赛事为主轴，后续技术章节作为可交叉引用的附录。

视觉风格：日系可爱风（マカロン粉彩、圆角贴纸感、Q 版赛车插画），中英双语文案（中文为主、英文小字辅注）。

## About the Design Files
本包中的 `F1 赛道年代记.dc.html` 是 **HTML 设计参考/可交互原型**，展示预期的外观与交互，**不是可直接上线的生产代码**。开发任务是在目标代码库现有环境（React / Vue / 小程序 / 原生等）中按其既有模式**重新实现**这些设计；若尚无环境，请自行选择最合适的技术栈（推荐 React + 移动优先布局）。文件为自定义组件格式（模板 + 一个 React 风格 class），可直接在浏览器打开体验，也可读取源码作为逻辑参考。

`image-slot.js` 是原型用的拖拽图片占位组件，生产中应替换为真实素材/上传方案。

## Fidelity
**High-fidelity（高保真）**：颜色、字体、间距、圆角、投影、交互均为最终设计意图，请按像素级还原（数据除外，见「数据完备度」）。

## 数据完备度（重要）
原型数据是「骨架完整、细节分层」的：
- 全部 76 个赛季（1950–2025）的：冠军车手（中/英文名）、冠军车、关键技术标签均为真实数据
- **1988 赛季为「完整数据样板」**：16 场分站赛（冠军+冠军车）、车手榜 Top3、18 辆参赛车全清单、3 个技术详情（动画/图解/3D 三种形式）
- 12 辆名车、12 项技术、10 位车手的博物馆条目含完整档案
- 其余年份的分站赛/车手榜/参赛车显示「资料整理中」占位卡
- 正式开发时应接入完整数据源（如 Ergast/Jolpica F1 API），按 1988 样板的结构铺满所有年份

## Screens / Views

### 1. 主屏：赛道时间轴（Home）
- **布局**：固定 header + 年代 chips 条 + 可滚动赛道区。设计宽度 390px（手机），整条路总高约 8000px
- **Header**：左侧 logo「GRAND PRIX 図鑑」（Baloo 2 800 20px #3a3532，「図鑑」为 #ff8fab）+ 副标题（9.5px #a89e94）；右侧一个「🏛️ 博物馆」入口按钮（白底 #fff、2px #f0e5da 边框、圆角 13px、高 38px）
- **年代 chips**：'50s–'20s 共 8 个胶囊（Baloo 2 800 11px）。当前年代：以年代色填充、白字；非当前：白底灰字 opacity .55。点击平滑滚动到该年代；**随滚动自动横向滚动居中并高亮**（transition all .3s）
- **赛道**：SVG path，路面 #4a4440 描边 34px、中央白色虚线（3px，dasharray 10 14，opacity .8）。路从顶部中央起，左右蛇形（左 x=64 / 右 x=326 交替）
- **起点横幅**：格子旗条（repeating-conic-gradient 8px 格）+ 深色圆角横幅「1950 発車! START / 银石 · 第一届世界锦标赛」
- **年份节点**（路上）：普通年 24px 白底圆点、4px 年代色描边；传奇年 30px、#ffd23e 底、#3a3532 描边。均可点击
- **普通年份卡**（170px 宽小卡）：白底、2px #f0e5da 边、圆角 14px、投影 0 3px 0 #f0e5da。内容：年份（Baloo 2 800 15px 年代色）+ 冠军名（10px 900 #3a3532）+ 👑 + 冠军车（9px #a89e94）+ 可选技术 tag（8.5px，#fff3d6 底 #c78a2d 字胶囊）。点击 → 原地弹出 popover
- **传奇年份大卡**（218px，共 14 个：1950/57/59/68/76/78/88/92/94/2004/09/14/21/25）：渐变白底、年代色半透明边框、顶部悬浮「★ 徽章」（#3a3532 底 #ffe3ec 字）、大年份数字（26px）、标题行、冠军+车、技术胶囊、「进入该赛季 ▸」、右下 Q 版赛车 SVG（floaty 3s 上下浮动动画）。点击 → 进入年份详情
- **透明度联动**：所有年份卡默认 opacity 0.42；当行驶小车 y 距该卡节点 <190px 时变为 1（transition opacity .4s）
- **行驶小车**：俯视 Q 版粉色赛车 SVG（26×44），固定于视口中部对应的路面位置；随滚动沿 path 移动并按切线转向（转角限制 ±60°，车头永远朝前进方向）。scroll 事件 + requestAnimationFrame 节流
- **年代横幅**：每 10 年一条（96px 高），小格子旗 + 白色胶囊：年代（Baloo 2 800 15px 年代色）+ 中文主题 + 英文小字。pointer-events:none
- **年份 popover**（点普通年）：238px 白卡、年代色边框、popin .16s 弹出动画；含年份、冠军（中英）、冠军车、技术 tag、「进入该赛季 GO! ▸」主按钮（#3a3532 底 #ffd23e 字）、✕ 关闭
- **终点**：格子旗 + 「🏁 驶抵现在 · 2026 新规元年，待续…」

**年代色板**（ERAS）：
- 1950s #b98ad6 前置引擎的先驱年代 / 1960s #5bbcd4 中置革命与车库军团 / 1970s #f2955d 空气动力学觉醒 / 1980s #e0527e 涡轮狂飙时代 / 1990s #3d9c7b 电子与安全革命 / 2000s #c78a2d V10 最强音 / 2010s #5b7fd4 混动与数据时代 / 2020s #e8543f 地面效应新纪元

### 2. 年份详情页（全屏覆盖，popin .18s）
- **Header**：← 返回、年份 +「シーズン」（#ff8fab）、SEASON n/76、右侧「🔊 引擎声」按钮（#ffe3ec 底 #e0527e 字，触发 WebAudio 合成引擎轰鸣：双 sawtooth 振荡器 65→520→140Hz + lowpass 1200Hz，约 1.7s）
- **冠军车 hero 台**：天蓝渐变舞台（#cfe8ff→#e9f4ff，68% 处切白），伪 3D 车模：pointer 拖动改变 rotateY（perspective 700px，±45° 限制），左上「◀ 拖动旋转 ▶」徽章，右上年代名。车下方车名（Baloo 2 800 16px）+ 副标题
- **冠军条**：#3a3532 深色圆角条，👑 圆徽 + 「WORLD CHAMPION」（#ffd23e 8.5px 字距 2px）+ 冠军中文名 + 英文小字
- **参赛车图鉴**：横向滚动卡列（128px/张，白底，冠军车 #ffd23e 描边 + 👑 角标），每张：Q 版侧视车 SVG（按车队涂装色）、车名、车队·车手、「查看细节 ▸」。点击 → 车辆详情页。无完整数据年份：1 张冠军车 + 虚线「资料整理中」占位卡
- **分站赛**：白色圆角列表容器，**默认显示前 3 站**，底部「ALL · 查看全部 16 站 ▾」展开 /「收起 ▴」。每行：轮次徽章（30×22 圆角，普通 #f5ece1/名场面 #ffd6e3）、国旗 emoji、「XX大奖赛」+ 名场面红字注记、🏆 冠军车手、右侧冠军车胶囊（名场面行粉色高亮）
- **车手榜 Top3**：白卡列表，行：奖牌方块（金 #ffd23e/银 #e9e2da/铜 #f5d9c3 + 对应深字色）、中文名+英文·车队、积分（Baloo 2 800 15px，第一名 #e0527e）、行尾 ▸。**整行可点 → 车手详情页**
- **本季技术**：2–3 张技术卡（高亮 #fff3d6/#f2dfae、普通 #dff3ec/#b9e2d2），右上形式角标（▶ 动画 / 🔍 图解 / 🧊 3D / 📖 图文，#3a3532 底白字），icon + 标题 + 副题 + 等级胶囊（进阶：#3a3532 底 #ffd23e 字；科普：白底绿字）。点击 → 技术详情页
- **上一年/下一年**：两个大按钮（白 / 粉 #ffe3ec）

### 3. 技术详情页（全屏覆盖）
Header：← 返回、icon+名称、年份·副题、等级胶囊、形式徽章。内容区按形式分四种：
- **▶ 动画**（例：涡轮增压）：#3a3532 深色舞台，SVG 动画：废气流（#f2955d 管道 + 流动虚线 animate stroke-dashoffset 0.9s）→ 旋转涡轮（4 叶 #f2b63c，CSS spin 1s linear infinite）→ 同轴 → 压气机（#7ec8e8，同转速）→ 高压空气流；下方 BOOST 增压条（@keyframes boost：宽 16%→92%，1.4s alternate 无限）
- **🔍 图解**（例：低趴姿态）：天蓝舞台，双车侧视对比 SVG + 虚线引出标注卡（重心高·迎风面大 / 车身更低更贴地），底部结论句
- **🧊 3D**（例：Honda RA168E）：深色舞台，可拖转（rotateY）分解引擎模型：V6 缸体 + 左右缸盖（floaty 错峰浮动）+ 旋转涡轮 + 泵组 + 虚线装配线，底部 3 个数据点（685hp / 150L / 2.5bar）
- **📖 图文**（其余技术）：拖入历史照片的 image slot（130px）
共用底部：**讲解卡**（正文 11px #6b6259 行高 1.75 + 「🎧 语音讲解/⏹ 停止」按钮，Web Speech API zh-CN rate .95）+ **时间轴定位卡**（「1988 ↩」粉色按钮 → 关闭详情、滚动主屏赛道到该年、普通年自动展开 popover）

### 4. 车辆详情页（全屏覆盖）
Header：← 返回、车名、车队·年份、「🔊 引擎声」。内容：伪 3D 可拖转车模舞台（含「3D ぐるぐる · 拖动旋转」徽章、冠军车加「👑 冠军车」金徽）；规格表白卡（车队/车手/引擎/分站胜利，虚线分隔行）；一句话车史；「物料」区：3D 模型卡（绿色，已启用）+ 视频占位卡（虚线）+ 两个可拖入实车照片的 image slot（112px）。注：知名车型后续应替换为真实 3D 模型（如 glTF + three.js），Q 版 SVG 为过渡方案

### 5. 车手详情页（全屏覆盖，无 3D）
Header：← 返回、🪖 车手名、英文名、「👑 ×N」冠军数徽章（#ffd23e）。内容：左照片槽（128×150 image slot）+ 右规格卡（世界冠军年份/效力车队/标签）；「车手故事 STORY」正文卡 + 🎧 语音讲解；「代表性赛季」定位卡（年份 ↩ 回赛道）

### 6. 博物馆（底部抽屉，sheetup .22s，遮罩 rgba(58,53,50,.35)）
圆角 26px 顶、拖动把手、标题「🏛️ 博物馆 MUSEUM」、✕。三个 tab 胶囊：🏎️ 名车 / 🔧 技术 / 🪖 车手（选中 #ffd6e3 底 #ffb7cd 边 #e0527e 字）。条目卡：**整行可点进入对应详情页**（标题旁 ▸），右侧粉色「年份 ↩」按钮定位回赛道（stopPropagation）。数据：名车 12 / 技术 12 / 车手 10

## Interactions & Behavior 汇总
- 滚动主屏 → 小车沿 path 行驶+转向、年份卡近处显影、年代 chip 高亮+自动居中（rAF 节流）
- 年代 chip 点击 → smooth scroll 至年代横幅
- 普通年点击 → popover；传奇年/GO 按钮 → 年份详情
- 伪 3D 拖转：pointerdown/move/up + setPointerCapture，rotateY = 初始 + Δx×0.35，clamp ±45°
- 引擎声：WebAudio 合成（见上）；语音讲解：speechSynthesis，切页/返回时 cancel
- 所有覆盖页返回按钮需记住来源（年份页 vs 博物馆）
- 分站赛展开/收起状态在切换年份时重置为收起

## State Management
- `view`: 'home' | 'year' | 'car' | 'tech' | 'driver' | 'appendix'
- `vy`(当前年份), `expanded`(popover 年份), `heroRot`(3D 角度), `tab`(博物馆 tab), `car`{x,y,a}(小车位置), `curDec`(当前年代), `carSel`/`techSel`/`driverSel`(详情对象), `backTo`(返回目标), `racesAll`(分站展开), `speaking`(TTS 中)
- 数据获取：赛季列表可静态打包；分站/车手榜/参赛车建议按年懒加载
